1：按钮悬浮在页面上，不随页面的滚动而动
<div id = "login" style = "position:fixed; top:50px;">
       <a href="//login">登录</a>
</div>
<script>
    $(document).ready(function(){
        $("#login").css("bottom", 0);
        $("#login").css("right", 0);
        $("#login").css("position", "fixed");
    })
</script>
这里用到了jQuery，也可以使用js实现。
